Instructions for Navigation

Navigation

  • To navigate, there is a navigation panel on left hand side.
  • Click on 'Home' to come to this screen.
  • Click on 'Contacts' to view all the contacts.
  • Click on 'Sent Messages' to view all sent messages in reverse chronological order.  

Pertinent Details

  • Contacts are arbitrarily created at the start of application.
  • All the contacts are stored in an Entity "contacts" in Database named "sms".
  • Fields in Entity are: Id (primary-key), FirstName, LastName, Mobile.

Walk-Through

  • To see all contacts, Click 'Contacts' from left-navigation-bar.
  • To send a message, select a contact from the list.
  • You may edit editable contact details, which will be reflected in database.
  • To send a message, Click 'Send-Message' Button.
  • Type your message and send.
  • OPT is automatically generated as soon as you open 'Send-Message' page.

Limitations

  • There may be delay in delivery of OTP message.
  • Trial Accounts can not send messages to unverified numbers using Twilio.

Spring-Boot-Application

  • IDE: Spring-Tool-Suite
  • Front-End: Angular-JS & Javascript
  • Back-End: MySql
  • SMS-Gateway: Twilio
